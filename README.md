# openclaw

This is generic openclaw in a container for agent development.

## cheliped

This is the name of the user in the container

The large, crusher claw on a lobster is called the cheliped (or more
specifically, the large claw is the crusher claw, while the smaller, sharper
one is the cutter claw). Together, these claws are known as
chelae (singular: chela). The crusher claw is used to crack open hard-shelled
prey, while the cutter claw is used for tearing food apart.
